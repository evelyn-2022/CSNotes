## Operations

### Plus `+=` \ Minus `-=`
```
x = 1
x += 1
print(x)
```
*output:*
```
2
```
<br>

### Exponential `**`
```
print(3**2)
```
*output:*
```
9
```
<br>

### Remainder `%`
```
print(38 % 3)
```
*output:*
```
2
```
<br>

### Integer Division `//`
```
print(22//7)
```
*output*
```
3
```

## Functions

### `int()` Function
Can be used to round a floating-point number <u>down</u>. 
```
print(int(7.7))
```
*output:*
```
7
```
==Compare with `round()`==
```
print(round(7.5))
```
*output:*
```
8
```
<br>

`abs()` Function
Return the <u>absolute value</u> of the argument.
```
print(abs(-1))
```
*output:*
```
1
```
