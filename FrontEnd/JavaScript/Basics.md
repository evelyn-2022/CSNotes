## Basics

### Open developer tool

`ctrl + shift + J`

### Where to put .js file link

`script` link should be put **at the end of body tag**

### Clear console

clear console: ctrl + k

## Debugger

In console, enter:

```js
debugger;
functionName(params...);
```

The debug mode will be opened.
